------------------------------------------------
full-screen image viewer and slideshow in Python
------------------------------------------------

Requirements:

 * Python 2.7+ or 3.2+
 * python-imaging (PIL/Pillow)
 * python-tk
 * python-imaging-tk

Example::

    $ git clone https://github.com/cgoldberg/easymiv.git
    $ cd easymiv
    $ ./easymiv.py /home/cgoldberg/images/

Keyboard Controls:

* q : quit
* <Escape> : quit
* <Control-c> : quit
* <space> : show next
* <Right> : show next
* <Down> : show next
* <Return> : show next
* <Left> : show previous
* <Up> : show previous
* s : slideshow
* Function Keys F1-F12 : slideshow
