------------------------------------------------
full-screen image viewer and slideshow in Python
------------------------------------------------

Requirements:

* Python 2.7+ or 3.2+
* python-imaging (PIL/Pillow)
* python-tk
* python-imaging-tk

Example::

    $ git clone https://github.com/cgoldberg/easymiv.git
    $ cd easymiv
    $ ./easymiv.py /home/cgoldberg/images/

Keyboard Controls:

* Quit:
 * q
 * <Escape>
 * <Control-c>
   
* Show Next:
 * <space>
 * <Right>
 * <Down>
 * <Return> : show next

* Show Previous:
 * <Left>
 * <Up>
